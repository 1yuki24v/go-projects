# HackType — Lightweight Recon Tool in Go

**HackType** is a fun, interactive cybersecurity recon tool written in **Go**.  
It’s designed for beginners to learn Go while building a practical tool.

---

## Features

- Ask for a target domain in the terminal  
- Find the IP address of the target  
- Scan common ports (`80`, `443`, `22`, `8080`)  
- Get HTTP server info and status code  
- Runs port scans concurrently for faster results  

---

## Usage

Run the tool:

```bash
go run hacktype.go
